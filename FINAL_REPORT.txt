## Owner Action Required

### 1. Discord OAuth (Required for Web Dashboard Login)

1. Go to https://discord.com/developers/applications
2. Open your bot application
3. Go to OAuth2 settings
4. Copy Client ID and Client Secret
5. Add to .env:
   ```
   DISCORD_OAUTH_CLIENT_ID=<your_client_id>
   DISCORD_OAUTH_CLIENT_SECRET=<your_client_secret>
   ```

### 2. Domain (Optional for HTTPS)

If you have a domain:
1. Update in .env:
   ```
   WEB_PUBLIC_URL=https://YOUR_DOMAIN
   WEB_COOKIE_SECURE=true
   WEB_CORS_ORIGINS=https://YOUR_DOMAIN
   DISCORD_OAUTH_REDIRECT_URI=https://YOUR_DOMAIN/api/v1/auth/discord/callback
   ```
2. Configure SSL certificate
3. Update nginx config

## VPS Deployment (Run on VPS)

